## ✨ What This Agent Does

- Automatically searches for potential clients based on your target criteria
- Scrapes and enriches lead data (name, company, job title, activity)
- Uses AI to qualify leads and assign a score
- Generates natural, personalized LinkedIn or email messages
- Saves all leads + ready-to-send messages into Google Sheets
- Supports daily scheduled automation

## 🚀 How to Use This Agent

1. Click the Gumloop link above and **Duplicate** the agent
2. Connect your Google Account for Sheets integration
3. Update the target audience keywords and your service description
4. Run the workflow daily
5. Review leads and messages before outreach
